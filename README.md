# Dotfiles
This is a collection of dotfiles for Zsh, Nvim and Tmux with other tools. Mi casa es su casa.

## Setup
Move my kick ass zsh theme to Oh My Zsh's theme folder:
```
cp pollele.zsh-theme ~/.oh-my-zsh/themes/pollele.zsh-theme 
```

**Tips:**
Use symlinks to keep dotfiles and your dotfile repo in sync.
